# Paraphrase Mining Using Sentence Transformers

## Overview
This project demonstrates how to perform **paraphrase mining** using the **Sentence Transformers** library in Python. It generates sentence embeddings using the pre-trained **all-MiniLM-L6-v2** model and identifies pairs of sentences that are semantically similar based on cosine similarity.

The notebook provides a practical introduction to semantic similarity and Natural Language Processing (NLP), making it suitable for beginners and students exploring text analysis.

---

## Features
- Generate sentence embeddings using Sentence Transformers
- Perform paraphrase mining on a collection of sentences
- Measure semantic similarity using cosine similarity
- Identify the most similar sentence pairs
- Simple and easy-to-understand implementation

---

## Technologies Used
- Python
- Sentence Transformers
- PyTorch
- NumPy
- Google Colab / Jupyter Notebook

---

## Model Used
- **all-MiniLM-L6-v2**

This lightweight transformer model converts sentences into dense vector embeddings that capture semantic meaning, enabling efficient similarity comparisons.

---

## Project Workflow
1. Import required libraries.
2. Load the pre-trained Sentence Transformer model.
3. Define a collection of sentences.
4. Generate sentence embeddings.
5. Apply paraphrase mining to compare sentence pairs.
6. Display the most similar sentence pairs along with their similarity scores.
